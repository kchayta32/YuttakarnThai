Game Overview

ภาพรวมเกม

เกมกลยุทธ์เรียลไทม์ที่พาคุณย้อนรอยประวัติศาสตร์การทหารไทย

📜

ความแม่นยำทางประวัติศาสตร์

เครื่องแบบ อาวุธ และยุทธวิธีอ้างอิงจากพิพิธภัณฑ์กองทัพไทย พร้อม "Museum Mode" ให้ความรู้เพิ่มเติม

⚔️

วิวัฒนาการสงคราม

จากช้างศึกและดาบในปี 1547 สู่ยุคโดรนและยานเกราะในปี 2025 กลไกเปลี่ยนตามยุคสมัย

🎯

กลยุทธ์เชิงลึก

ระบบ Rock-Paper-Scissors, ภูมิประเทศให้ข้อได้เปรียบ, ระบบขวัญกำลังใจ และการจัดการทรัพยากร

🎓

คุณค่าทางการศึกษา

เหมาะสำหรับผู้ชื่นชอบประวัติศาสตร์และการศึกษาในระดับมัธยมถึงมหาวิทยาลัย

Executive Summary

RTS: Yutthakan Thailand เป็นเกมกลยุทธ์เรียลไทม์ผู้เล่นคนเดียว ที่พาผู้เล่นผ่านความขัดแย้งทางทหารและการเมืองที่สำคัญ 9 ครั้งในประวัติศาสตร์ไทย ตั้งแต่ศตวรรษที่ 16 จนถึงสถานการณ์สมมติในอนาคตอันใกล้ เกมเน้นความสมจริงเชิงยุทธวิธี ความถูกต้องทางประวัติศาสตร์ และการจัดการทรัพยากรเชิงกลยุทธ์

Game Systems

ระบบเกม

กลไกหลักที่สร้างประสบการณ์การเล่นที่ลึกซึ้ง

Resource Bar - ข้าว, เสบียง, เชื้อเพลิง
Minimap - แผนที่ขนาดเล็ก
Unit Info - ข้อมูลหน่วย
Commands - คำสั่งต่างๆ

🔄 Core Game Loop

  1. Briefing - รับบริบททางประวัติศาสตร์และภารกิจ
  2. Base/Army Management - รวบรวมทรัพยากร สร้างอาคาร ฝึกหน่วย
  3. Exploration - สำรวจแผนที่ เปิดหมอกสงคราม ยึดจุดยุทธศาสตร์
  4. Combat - ต่อสู้ศัตรูด้วยระบบ Counter และภูมิประเทศ
  5. Victory/Loss - บรรลุหรือล้มเหลวในภารกิจ

💰 ระบบทรัพยากร

🌾
อาหาร/ข้าว

หน่วยชีวภาพใช้อาหาร เก็บจากฟาร์มหรือนาข้าว

📦
เสบียง/ทอง

อุปกรณ์/อาวุธใช้เสบียง เก็บจากจุดค้าขายหรือการส่งเสบียง

⛽
เชื้อเพลิง (ยุคสมัยใหม่)

ยานพาหนะต้องการเชื้อเพลิง เก็บจากคลังน้ำมัน

⚔️ ระบบการต่อสู้

Damage FinalDamage = BaseDamage × (100 / (100 + TargetArmor)) × RangeMultiplier
Accuracy HitChance = BaseAccuracy - (Distance × FalloffFactor) - CoverBonus

ระบบขวัญกำลังใจ (Morale)

  • < 50% Morale = -20% อัตรายิง/ความเร็ว
  • 0% Morale = หน่วยหนีไม่สามารถควบคุมได้

👁️ ระบบการมองเห็น

Shroud - ยังไม่สำรวจ
Fog - สำรวจแล้วแต่ไม่เห็นปัจจุบัน
Visible - มองเห็นชัดเจน

หน่วยบางประเภท (กองโจร, สไนเปอร์) ซ่อนตัวในป่าจนกว่าจะโจมตีหรือถูกตรวจจับ

Key Features

คุณสมบัติเด่น

สิ่งที่ทำให้ RTS: ยุทธการไทย โดดเด่น

📚

Museum Mode

โหมดพิพิธภัณฑ์ให้ความรู้เพิ่มเติมเกี่ยวกับประวัติศาสตร์ เครื่องแบบ และอาวุธจริง

🌍

Localization

รองรับหลายภาษา (TH/EN/CN) สามารถเปลี่ยนได้ทันทีในเกม

👁️

Accessibility

UI Scale ปรับได้ 80%-150%, โหมดสำหรับผู้บกพร่องการมองเห็นสี

🎵

Adaptive Music

เพลงประกอบปรับตามยุคสมัย - ดนตรีไทยโบราณถึงออเคสตราสมัยใหม่

🎮

Control Groups

เลือกหน่วยด้วย Box select, Double click, Ctrl+1-9

💾

Save/Load

ระบบบันทึก/โหลดเกมด้วย JSON serialization
